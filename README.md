### ChainVoice
> *Stopping duplicate invoice fraud — before it reaches the lender*

A **hybrid blockchain system for MSME invoice financing** built for DeltaBuild 2k26. Integrates HyperLedger Fabric + IPFS to generate SHA-256 cryptographic invoice fingerprints for real-time cross-lender fraud prevention.
